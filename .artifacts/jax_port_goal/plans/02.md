## Oracle Contract

`fixed_scalar` and `fixed_gradient_vjp`. The oracle is the closed-form
circular-coil B_self formula at the centroid:

```
B_z = mu_0 * I / (4 * pi * R0) * (log(8 * R0 / a) - 3/4)
```

This is the limit of the Hurwitz / Landreman / Antonsen Eq. (98)
regularized self-field for a circular coil with circular cross-section
of radius `a` in the `x-y` plane. The `B_x` / `B_y` components must
vanish at machine precision.

Upstream SIMSOPT at `1b0cc3a96063197cdbdd01559e04c25456fbe6ff` ships the
identical kernel and is the parity oracle for the kernel implementation.
